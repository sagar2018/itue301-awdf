About This Course

ITUE301AWDF is a lab-only course that teaches modern full-stack web development using industry-standard technologies. Over 13 weeks, students build a complete, production-ready Task Management Application incrementally adding features each week from UI components through to containerised deployment and AI integration.

Each two-hour session follows a structured format: warm-up, core lab work, progress check, and wrap-up. Practical marks are auto-graded via GitHub Actions; assignment marks combine automated and manual evaluation.

Project Thread Overview

Technology Stack

  •   React 18 component-based UI
  •   React Router v6 client-side navigation
  •   Node.js + Express.js REST API server
  •   MongoDB + Mongoose NoSQL database
  •   Redis server-side caching
  •   JWT stateless authentication
  •   Docker + Docker Compose containerisation
  •   GitHub Actions CI/CD & auto-grading
  •   Jest unit & integration testing
  •   Tailwind CSS utility-first styling
  •   OpenAI API AI feature integration
  •   Git / GitHub version control

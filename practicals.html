PhaseOptimization

Topics Covered

  • Adding in-memory caching to the Task Management backend using node-cache
  • Caching the GET /tasks response with a 60-second TTL
  • Cache invalidation on POST, PUT, DELETE write operations
  • Measuring and recording API response times with Postman (3 readings each, cached vs uncached)

Coursera Module Practical Handout
Teaching Guide Tell students to record uncached response times BEFORE adding the cache check. Walk through a live example of forgetting cache invalidation to make the risk concrete.
Common Mistakes Forgetting to invalidate cache after write operations; setting TTL so short the cache never gets hit; taking only one reading instead of multiple samples.
Rubric (20 marks) node-cache Setup (4) · Cache on GET (5) · Cache Invalidation (5) · Response Time Comparison (6)

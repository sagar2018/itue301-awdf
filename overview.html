Course Outcomes (CO)

At the end of this course, students will be able to:

CO Statement Bloom's Level
CO1 Design and Implement component-based frontend systems using reactive state management and performance optimization strategies for scalable web applications. Apply (L3)
CO2 Design and Implement secure RESTful backend services incorporating middleware pipelines, validation layers, and caching mechanisms. Apply (L3)
CO3 Select and evaluate appropriate data modeling, migration, encryption, and storage strategies based on workload characteristics and consistency requirements. Analyse (L4)
CO4 Design scalable web system architectures by applying stateless service design, load balancing, asynchronous processing, and performance profiling techniques. Analyse (L4)
CO5 Implement cloud-native deployment pipelines using containerization, CI/CD automation, and deployment strategies while monitoring reliability and resource optimization. Apply (L3)
CO6 Apply security engineering principles and responsibly integrate AI-based services into web systems while analyzing ethical, privacy, and governance implications. Apply (L3)
